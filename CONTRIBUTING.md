# Contributing to "Keyple Plugin CNA Flowbird Android Java Lib" implementation

Thanks for your interest in this project.

## Contributing

1. [Fork](https://help.github.com/articles/fork-a-repo) the [calypsonet/keyple-plugin-cna-arrive-android-jvm-lib](https://github.com/calypsonet/keyple-plugin-cna-arrive-android-jvm-lib) repository
2. Clone repository: `git clone https://github.com/[your_github_username]/keyple-plugin-cna-arrive-android-jvm-lib.git`
3. Create your feature branch: `git checkout -b my-new-feature`
4. Make your changes
5. Commit your changes: `git commit -m "Add some feature" -s`
6. Push feature branch: `git push origin my-new-feature`
7. Submit a pull request
